General Electronic Module Tester (GEMT)

Overview

Click the image above to go to the Github Repo!

GEMT is an open-source library that allows you to easily create a testing menu for your IOT and robotic electronic modules using the Arduino framework. With the GEMT, you can define your own test functions, set up menus, and execute tests with a simple user interface.
